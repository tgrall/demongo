To calculate stats, for example numbers of tweets by seconds it a simple aggregate query, with a group. The aggregate pipe is divided in 2 main phases:

  1. Do a projection that emit the date fo creation of the tweet 'created_at' in various fields (year, month, day, hour, mn, sec)
  2. Do a group on this date and count the number of tweet ({'$sum':1})
Note: this query is nice to show the type of things that could be done with the Aggregation Framework, but it is more efficient to take another approach when dealing with such aggregation, take a look to the Pre-Aggregated Reports use case in MongoDB Documentation.
