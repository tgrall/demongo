MongoDB supports Full Text Search.

For this you need to:

  1. Enable Full Text Feature : 
    mongod --setParameter textSearchEnabled=true
  2. Create a Text index : 
    db.messages.ensureIndex( {'text':'text'})
    In this case on the field 'text' of the 'messages' collection
More information in the documentation.

Using MongoDB Full Text search you can use many operators to search specific messages, more information in the documentation.
