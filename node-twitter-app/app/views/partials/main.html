Demongo : See MongoDB in Action!


Discover the power of MongoDB query engine using a simple Twitter database!

Twitter User :

#demobash #mongodb #nosql

Technology

This application is based on the MEAN stack:
  • MongoDB : data storage, queries and aggregation
  • ExpressJS : REST layer
  • AngularJS : User Experience
  • Node.js : Application Server
and uses many features from MongoDB including:
  • Node.js Driver (Java for the injector)
  • Simple Queries
  • Full Text Queries
  • GeoSpacial Queries
  • Aggregator Framework
  • Create, Update ... operations
